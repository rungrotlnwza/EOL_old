# 📁 EOL PHP Project Structure

## 🔄 กระบวนการตรวจสอบและ Copy ไฟล์

**ขั้นตอนการทำงาน (10 ขั้นตอน):**

1. **รอรับ path ไฟล์** - รับชื่อไฟล์ PHP ที่ต้องการตรวจสอบ (เช่น `fn_office.php`)

2. **ตรวจสอบไฟล์ที่เกี่ยวข้อง** - วิเคราะห์ไฟล์เพื่อหา:
   
   **PHP Code:**
   - `include()` / `require()` → ไฟล์ PHP ที่ถูกเรียกใช้
   - `header("Location: ")` → PHP redirect
   
   **HTML:**
   - `<img src="">` → ไฟล์รูปภาพ
   - `<a href="">` → ลิงก์ไปยังไฟล์ .php อื่นๆ
   - `<form action="">` → Form submission
   - `<link href="">` → CSS files
   - `<script src="">` → JavaScript files
   - `<iframe src="">` → Embedded pages
   - `<audio src="">` / `<video src="">` → Media files (ของระบบ)
   
   **CSS:**
   - `background-image: url()` → รูพื้นหลังใน CSS/inline style
   
   **JavaScript:**
   - `fetch()`, `$.ajax()`, `XMLHttpRequest` → AJAX calls
   - `window.location`, `location.href` → JS redirect
   
   **Special:**
   - `www.engtest.net` URL → ถือเป็น internal link (ต้องดึง path มาเช็คด้วย)
   - `<meta http-equiv="refresh">` → HTML auto redirect

3. **ตรวจสอบไฟล์ใน old/** - ตรวจสอบว่าไฟล์ที่เกี่ยวข้องมีอยู่ใน `old/` หรือไม่

4. **ตรวจสอบไฟล์ใน php/** - เช็คว่าไฟล์ที่เกี่ยวข้องมีใน `php/` แล้วหรือยัง
   - ถ้ามีแล้ว → ข้าม
   - ถ้ายังไม่มี → เพิ่มเข้า .bat

5. **สร้างไฟล์ .bat** - สร้าง `copy_[ชื่อไฟล์]_files.bat` ที่มี:
   - คำสั่ง `mkdir` สร้าง folders
   - คำสั่ง `copy` คัดลอกเฉพาะ code และ assets
   - **ไม่รวม**: user data, database data, logs, cache, temp files
   - ใช้ `exit` แทน `pause`

6. **รันไฟล์ .bat** - Execute batch script เพื่อ copy ไฟล์

7. **อัพเดท eol.txt (ไฟล์หลัก)** - เพิ่ม ✅ ให้กับไฟล์หลักเท่านั้น (ไฟล์ที่ชื่อตรงกับ .bat)
   - ไฟล์ที่เกี่ยวข้อง (dependencies) ไม่ต้องใส่ ✅
   - เพิ่มไฟล์ใหม่ลงใน tree structure

8. **อัพเดท eol.txt (รายการสรุป)** - อัพเดท 2 ส่วน:
   - เพิ่มไฟล์หลักใน "ไฟล์ที่ตรวจสอบแล้ว"
   - เพิ่มรายการไฟล์ทั้งหมดที่ copy ใน "ไฟล์ที่ดาวน์โหลดแล้ว" (ระบุ path และจำนวนอย่างชัดเจน ไม่ใช้ ... หรือย่อ)

9. **สร้างคำสั่ง Git** - ให้คำสั่งสำหรับ copy:
   ```bash
   git add .
   git commit -m "ไฟล์ที่เกี่ยวข้องกับ [ชื่อไฟล์]"
   ```

10. **สรุปและรอไฟล์ถัดไป** - แสดงสรุปการทำงานและรอรับไฟล์ถัดไป

---

**หมายเหตุสำคัญ:**
- ✅ = ไฟล์ที่ตรวจสอบแล้ว (ตรงกับชื่อ .bat เท่านั้น)
- ไฟล์ library/helper (เช่น `paging.inc.php`, `connection.php`) ไม่ต้องสร้าง .bat
- `www.engtest.net` = internal link ต้องนับเป็นไฟล์ที่เกี่ยวข้อง

**สิ่งที่ต้อง Copy (SYSTEM):**
- ✅ Code: PHP, CSS, JS
- ✅ Assets: รูปภาพ UI/Design (logos, icons, buttons, backgrounds)
- ✅ Media: เสียง/วิดีโอของระบบ (ถ้ามี)
- ✅ การเชื่อมโยง: include, redirect, form, link, ajax

**สิ่งที่ไม่ต้อง Copy (USER DATA):**
- ❌ รูปภาพที่ user อัพโหลด (`user_images/`, `upload/`)
- ❌ บทเรียนที่ user สร้าง
- ❌ ไฟล์เสียง/วิดีโอที่ user upload
- ❌ Database data
- ❌ Logs, Cache, Temp files
